# 🔹 Scope kya hota hai?

👉 **Scope = kaha tak ek variable accessible hai (kahan use kar sakte ho)**

Simple:

> “Variable kis area me visible hai”

---

# 🔥 Types of Scope in JS

## 1️⃣ Global Scope

👉 Jo variable sab jagah accessible ho

```js id="g1l2ka"
let name = "Sumit";

function greet() {
  console.log(name);
}

greet(); // Sumit
```

✔ `name` har jagah use ho sakta hai

---

## 2️⃣ Function Scope

👉 Jo variable sirf function ke andar hi accessible ho

```js id="v8r4me"
function test() {
  let x = 10;
  console.log(x);
}

test(); // 10
console.log(x); // ❌ Error
```

✔ `x` sirf function ke andar hi hai

---

## 3️⃣ Block Scope (Important 🔥)

👉 `{ }` ke andar jo variable ho

```js id="1fuxg7"
if (true) {
  let a = 5;
  const b = 10;
}

console.log(a); // ❌ Error
console.log(b); // ❌ Error
```

✔ `let` aur `const` block scoped hote hain

---

## ❗ var ka scene (Important)

```js id="t6k5m9"
if (true) {
  var x = 100;
}

console.log(x); // 100 ✅
```

👉 `var` **block scoped nahi hota**
👉 wo function scoped hota hai

---

# 🔥 4️⃣ Lexical Scope (Advanced but easy 😎)

👉 Inner function outer function ke variables access kar sakta hai

```js id="o1twz9"
function outer() {
  let name = "Sumit";

  function inner() {
    console.log(name);
  }

  inner();
}

outer(); // Sumit
```

✔ Inner → outer ka access possible
❌ Outer → inner ka access nahi

---

# 🔥 Scope Chain

👉 JS variable ko dhundte waqt:

1. Current scope check karta hai
2. Phir outer scope
3. Phir global scope

```js id="0hvv34"
let a = 1;

function outer() {
  let b = 2;

  function inner() {
    let c = 3;
    console.log(a, b, c);
  }

  inner();
}

outer(); // 1 2 3
```

# 🔥 Quick Comparison

| Keyword | Scope           |
| ------- | --------------- |
| var     | Function scoped |
| let     | Block scoped    |
| const   | Block scoped    |

---

# 🔥 Common Mistake ❌

```js id="g6y9hc"
if (true) {
  var x = 10;
}

console.log(x); // 😬 unexpected access
```

👉 Isliye modern JS me:
👉 **`let` aur `const` use karo, `var` avoid karo**
