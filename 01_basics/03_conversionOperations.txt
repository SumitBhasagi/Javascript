Conversion Operations kya hote hain?

👉 Jab JavaScript ek data type ko dusre data type mein convert karta hai usse bolte hain Type Conversion

     📌 2 Types hote hain:
        1️⃣ Implicit Conversion (Automatic 😏)

          👉 JS khud se conversion karta hai

        2️⃣ Explicit Conversion (Manual 💪)

          👉 Tu khud conversion karta hai


🔥 1. Implicit Conversion (Type Coercion)

👉 JavaScript automatically type change karta hai  
    Example 1: console.log("5" + 2);

    Example 2: console.log("5" - 2);  

Yaad Rakhna-->  
  | Operation	      |         Result       |
  |     +	          |   String banata hai  |
  |   -, *, /	      |   Number banata hai  |
