Variables 

1. const : ek baar value assign ho gayi → change nahi kar sakte 
   
   Basic Syntax:   const name = "Sumit";  
   
   name ek variable hai
  "Sumit" uski value hai
   Aur ye value fix ho gayi

   const age = 20;
   age = 25;  // ❌ Error aayega // value change nahi krr sakte  

   Important Rule #1: Declaration ke time value deni hi padegi

   Important Rule #2: Objects aur Arrays ke saath twist ->

    const person = {
        name: "Sumit",
        age: 20
     };
     person.age = 21;   // ✅ Allowed

    But Pura Object replace nahi krr sakte --> person = { name: "Rahul" };  // ❌ Error
  
    Array example:  const arr = [1, 2, 3];
                  arr.push(4);   // ✅ Allowed
                  arr = [5, 6];  // ❌ Error

2. Let : let ek variable declare karne ka modern tarika hai -> Isme value change (reassign) kar sakte ho
         
         example: let age = 20;
                  age = 25;   // ✅ Allowed

    Rule#1: Redeclare nahi krr sakte 
          
          let name = "Sumit";
          let name = "Rahul";   // ❌ Error
          👉 Same variable dobara declare nahi kar sakte

3. Var : Ye old JavaScript ka variable keyword hai -> Aajkal use nahi karte (but interview ke liye jaana zaruri hai)
       
        example: var city = "Pune";
                 var city = "Mumbai";   // ✅ Allowed

                 Yaha Reassign bhi allowed , Redeclare bhi allowed 😬


  🔥 Main Difference: Scope: variable kaha tak accessible hai

    var = function scope
    let = block scope
