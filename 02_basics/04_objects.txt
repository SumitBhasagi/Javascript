++++++++ Built-In functions in Objects ++++++++++

1. combining two or more Objects into single Object 
   
    const obj1 = { 1: "a", 2: "b" };
    const obj2 = { 3: "a", 4: "b" };
    const obj4 = { 5: "a", 6: "b" };


  [1.1]   const obj3 = { obj1, obj2 }; // output --> 
          { obj1: { '1': 'a', '2': 'b' }, obj2: { '3': 'a', '4': 'b' } } 

           this will combine the objects into one object but they will be in nested/sub-objects manner
   
  [1.2]    const obj3 = Object.assign(obj1, obj2); // output --> 
           { '1': 'a', '2': 'b', '3': 'a', '4': 'b' }
 

  [1.3]    const obj3 = Object.assign({}, obj1, obj2, obj4);        
           output--> { '1': 'a', '2': 'b', '3': 'a', '4': 'b', '5': 'a', '6': 'b' } 

           Object.assign(target, sequence) --> This is the main syntax

           const obj3 = Object.assign({}, obj1, obj2, obj4); 

           so whatever is there in the sequence will merge into the target so to increase the readability of the object we add an empty object as a target followed by sequence of objects 

2. Accessing all the keys of an object 
       
          to do this thing we use the function .keys()

          Object.keys(obj_name);  --> syntax 
            
          console.log(Object.keys(tinderUser)); ,  
          output --> [ 'id', 'name', 'isLoggedIn' ]

          ye function object ke keys ka array banake return karta hain 

3. Accessing all the values only  
        
        to access the values obly we have this function named --> .values() 

        Object.values(obj_name) --> syntax
        
        console.log(Object.values(tinderUser)); 
        
        output --> [ '123abc', 'sammy', 'true' ]
        
        ye function object ke saare values ka array banake return karta hain ,

4. Object.entries(obj_name)  
         
         console.log(Object.entries(tinderUser));

         output --> [ [ 'id', '123abc' ], [ 'name', 'sammy' ], [ 'isLoggedIn', 'true' ] ]

         ye function object ke sare key-value pair ko single single array mein convert krke saare arrays ko ek aaray mein insert krke return krta hain, 

        Yaha pe 1st entry in array "key" rahegi fir 2nd entry uski "value"

5. obj_name.hasOwnProperty("property_name")  

        console.log(tinderUser.hasOwnProperty("isLoggedIn")); 

        yaha pe property_name mtlb object ka key name

        ye function ek boolean value return karta hain ki particular key uss object se belong krti hain yaa nahi ?


+++++++ DESTRUCTURING ++++++++

const course = {
  coursename: "js in hindi",
  price: "999",
  courseInstructor: "prakash sakaria",
}; 

 to access the properties of the objects we have 2 general methods which are
  1. course.courseInstructor
  2. course[courseInstructor]

assume there is a situation where we have to access the same property again & again then the code will get enlarge in number of lines and it is not good practices in JS

So to do such type of repitative access and reduce the redundancy of code we use Destructuring

this is how we do destructuring, here we can also rename a particular property with shotcuts
_______________________________________________
const { courseInstructor } = course; // here we are not renaming
const { courseInstructor: teacher } = course;

console.log(teacher);
