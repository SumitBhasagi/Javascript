# рЯФ• PART 2: Arrow Functions

## рЯФє Arrow function kya hota hai?

рЯСЙ Short syntax function likhne ka

```js

  return a + b;
};
```

рЯСЙ Shorter:

```js

```

---

# рЯФ• MAIN DIFFERENCE: `this` behavior вЪ†пЄП

рЯСЙ Arrow function me **apna `this` nahi hota**

рЯСЙ Wo parent scope ka `this` use karta hai

---

## рЯФє Example (Normal Function)

```js

  name: "Sumit",
  greet: function () {
    console.log(this.name);
  }
};

user.greet(); // Sumit
```

---

## рЯФє Example (Arrow Function вЭМ)

```js

  name: "Sumit",
  greet: () => {
    console.log(this.name);
  }

```

вЬФ Kyun?

* Arrow function ka `this` вЖТ global se aata hai
* object se bind nahi hota

# рЯФ• Arrow Function + this (Best Use)

# рЯФ• Key Differences

| Feature     | Normal Function          | Arrow Function               |
| ----------- | ------------------------ | ---------------------------- |
| this        | Dynamic (call pe depend) | Lexical (parent se aata hai) |
| arguments   | Available                | Not available                |
| constructor | Yes                      | No                           |
| syntax      | Long                     | Short                        |
