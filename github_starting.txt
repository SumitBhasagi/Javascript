# 🚀 FULL FLOW (Start → GitHub → Local → Push)

---

## 🟢 STEP 1: Create a repository on GitHub

1. Go to GitHub
2. Click **New Repository**
3. Name it: `javascript-notes` (or anything)
4. Keep it **Public**
5. ❗ DO NOT initialize with README (important)
6. Click **Create Repository**

---

## 🟢 STEP 2: Copy repository URL

You’ll see something like:

```
https://github.com/your-username/javascript-notes.git
```

Copy it.

---

## 🟢 STEP 3: Open your project folder in terminal

```bash
cd Desktop
mkdir Javascript
cd Javascript
```

---

## 🟢 STEP 4: Initialize Git

```bash
git init
```

👉 This makes your folder a Git repository.

---

## 🟢 STEP 5: Connect to GitHub repo

```bash
git remote add origin https://github.com/your-username/javascript-notes.git
```

---

## 🟢 STEP 6: Create your first files

```bash
touch index.js
touch notes.txt
```

OR manually create files:

* `index.js`
* `notes.txt`

---

## 🟢 STEP 7: Add files to staging

```bash
git add .
```

---

## 🟢 STEP 8: Commit your files

```bash
git commit -m "Initial commit - added JS files"
```

---

## 🟢 STEP 9: Set branch to main (important)

```bash
git branch -M main
```

---

## 🟢 STEP 10: Push to GitHub

```bash
git push -u origin main
```

👉 Now refresh GitHub → your files will be there 🎉

---

# 🔁 DAILY WORKFLOW (after setup)

Whenever you study JS:

```bash
git add .
git commit -m "Learned arrays"
git push
```

---

# ⚠️ If Git asks for login

Use:

* Username: your GitHub username
* Password: ❗ Use **Personal Access Token** (not actual password)

# 🧠 What you just learned (important)

* `git init` → start repo
* `git add` → select files
* `git commit` → save snapshot
* `git push` → upload to GitHub
